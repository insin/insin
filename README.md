  <strong>2024-12-02</strong> – <a href="https://github.com/insin/control-panel-for-twitter">control-panel-for-twitter</a> <a href="https://github.com/insin/control-panel-for-twitter/releases/tag/v4.4.0">v4.4.0</a>
  <div><em>Browser extension which gives you more control over your Twitter timeline and adds missing features and UI improvements - available for desktop and mobile browsers</em></div>
  <details><summary>v4.4.0 release notes</summary><p>Visit the <a href="https://jbscript.dev/control-panel-for-twitter" rel="nofollow">Control Panel for Twitter website</a> for installation links, more information about the extension, and FAQs. Follow <a href="https://twitter.com/ControlPanelFT" rel="nofollow">@ControlPanelFT</a> on Twitter for updates.</p>
<h2>Changes</h2>
<ul>
<li>The "Hide Grok" option now hides the new button which Twitter has added to all Tweets</li>
<li>Blanket hide all upsell links in your own profile - hides the current Premium upsell which will no doubt be replaced with another</li>
<li>Simplified hiding Views and bookmark buttons under Tweets, to avoid hiding the wrong button when Twitter changes things</li>
<li>Fixed detecting when we're on the Bookmarks page, as Twitter recently broke the page title</li>
<li>The Firefox version of Control Panel for Twitter now requires Firefox 121 or later</li>
</ul>
